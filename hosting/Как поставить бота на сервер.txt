1. Для начала создадим текстовый файл со всеми мудулями, библиотеками для того, что бы не писать их в ручную

- pip freeze # Просмотр всех установленных модулей, библиотек
- pip freeze > requiremets.txt # Создастся текстовый файл со всеми модулями, библиотеками

2. Воспользуемся сервером от timeweb.cloud

1) Облачные серверы -> создать сервер
2) Выбираем минимум Ubuntu 20.04
3) Регион Россия город близжайший
4) Бота с 2 ГГц хватит для бизнес бота 
5) Выбираем имя и оплачиваем бота

3. Настройка ключа SSH (Можно без него)

- ssh-keygen
- Выбор папки куда будет создаваться ключ для согласия папки по умолчанию Нажать enter
- Если ключ есть попросит перезаписать y/n на выбор
- Придумать секректную фразу (artem)
- Повторяем секретную фразу (artem)
- Ключ сгенерирован будет указано куда он сгенерирован
- cat путь/к/файлу # Для того, что бы посмотреть ключ, пишем команду cat с помощью нее можно зайти в любой файл и путь к файлу (Your public key '.pub')
- копирует ключ ssh-rsa начало и все остальное до конца .local
- Переходим в timeweb SSH-ключи -> добавить, вставляем ключ и Добавить (галочка не нужна)
- Сервер -> Доступ -> SSH-ключи -> ставим галочку на против нужного имени ключа (если не один)
- ssh root@ip.адрес.сервера # Далее в терминале пишем
- Вводим секретную фразу (artem)

4. Настройка сервера. Заходим на сервер через консоль на timeweb.cloud или через свою консоль если делал SSH

- Login : root
- Password : Доступ -> Root-пароль -> Скопировать
- sudo apt update # Установит утилиты которых не хватает
- sudo apt upgrade # Обновит эти утилит до последних версий
- sudo apt --reinstall install python3 или просто sudo apt --install python3 # Установка pythona
- y/s -> y
- sudo apt --reinstall install python3-pip или просто sudo apt --install python3 # Пакетный менеджер python и можно было устанавливать модули
- y/s -> y
- sudo apt install nodejs # Устанавливаем менеджер процессов для взаимодействия всеми процессами и управлением бота
- sudo apt install npm
- y/n -> y
- npm install pm2 -g

5. Заливаем бота на сервер (GitHub/FileZilla (client))

1) GitHub

- Переходим в консоль

- Login : root
- Password : Доступ -> Root-пароль -> Скопировать 

     - git clone https://github.com/Artem200023/BarbershopBot/tree/main.git
     - pwd # Корневой каталог
     - ls # Посмотерть файлы 
     - cd имя_папки/ # Переходим в папку
     - pip3 install -r requirements.txt # Устанавливаем все модули которые прописаны в этом файле (В папке с файлами бота)

- Запускаем бота (Нужно быть в папке с файлами бота)
    - pm2 start Barberbot.py -- interpreter=python3 # Запустить бота
    - pm2 stop Barberbot.py # Остановить бота

Если работаем в SSH не забываем прописать exit 
    - exit # Возвращаемся в свое устройство

     Хост : ip.адрес.сервера

     Порт : 22 (Это стандартный порт для SSH)
